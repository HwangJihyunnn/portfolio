INTBRIDGE ㅣ 인트브릿지

  • 페이지설계 : 반응형

  • 제작기간 : 2024.01~2024.01 1달 / 2명

  • 웹 크기 : max-width: 1920px

    태블릿 크기 : max-width: 720px

  • 프로젝트 개요 :

    기존에 있던 단조로운 자사홈페이지를 요즘 IT 아이덴티티에 맞게 블루와 퍼플컬러로 포인트를 주었으며 피그마를 이용하여 먼저 UIUX 레이아웃을 설계하였고 기업에서 추구하는 방향성과 자사 제품인 솔루션 제품을 강조하였습니다. 구조 설계 하나씩 전부 컨펌받은 뒤, 디자인쪽을 맡은 팀원과 서로 분담하여 성공적으로 기업 홈페이지를 만들 수 있었습니다. 또한, 기업 연혁 서브페이지 같은 경우 써보지 않은 JSON을 사용해보면서 JSON에 대한 구조를 알게되었습니다. 클론 코딩이 아닌 디자인부터 콘텐츠 내용 및 구조까지 직접 잡아가며 성장할 수 있는 계기가 되었습니다.

OAK VALLEY ㅣ 오크밸리

  • 페이지설계 : 반응형 Main + Sub*3(소개/다이닝/객실)

  • 제작기간 : 40시간 퍼블리싱 100%

  • 웹 크기 : max-width: 100%

    태블릿 크기 : max-width: 1024px

  • 프로젝트 개요 :

    오크밸리 사이트를 클론 코딩하면서 자바스크립트와 제이쿼리 라이브러리를 활용한 포트폴리오 사이트를 구현해보았습니다. 메인의 슬라이드 기능은 플러그인을 사용하지않고 직접 구현하였으며, 스크립트로 구현할 수 있는 부분은 최대한 직접 만들어보았습니다.

Center ㅣ 세상의 중심, 센터

  • 페이지설계 : 모바일 Main + Sub*3 + 카카오로그인 + 글쓰기 및 삭제 수정

  • 제작기간 : 5일 4명 / 퍼블리싱 25%

  • 모바일 크기 : max-width: 430px

  • 프로젝트 개요 :

    파이어베이스를 활용하여 이메일 로그인 구현과 게시글 삭제 및 수정 업로드를 하였으며 또한 카카오 api를 이용하여 카카오 로그인까지 구현하였습니다. 익숙치 않은 플러그인이였지만, 미처 몰랐던 부분이나 놓쳤던 부분도 챙기고 배우면서 기능적인 부분뿐만 아니라 의사소통 하는 법을 배우고 조금 더 성장하는 계기가 되었습니다.

    본 화면은 모바일 430px크기에 최적화 되어있습니다.

Giordano ㅣ 지오다노

  • 페이지설계 : Main

  • 제작기간 : 1주일 퍼블리싱 100%

  • 웹 크기 : max-width: 100%

  • 프로젝트 개요 :

    react를 이용하여 심플하게 지오다노 쇼핑몰을 만들어보았습니다. 익숙치 않은 언어로 처음 만들어보면서 컴포넌트는 독립적으로 작동하며 재사용이 가능하다는 장점을 알게되었습니다. 아직은 사용하는 것이 많이 서툴지만 계속 배워나가겠습니다.

Oh KitChen ㅣ 오키친

  • 페이지설계 : Web Main + Sub*3(인기레시피/신규레시피/계량팁)

  • 제작기간 : 1주 퍼블리싱 100%

  • 웹 크기 : max-width: 100%

  • 프로젝트 개요 :

    html/css 기본을 다져보고자 만들어본 프로젝트였습니다. 자바스크립트 기능보다는 전체 구조와 css를 다져보는데 집중해서 만들었습니다. 기능적인 요소보다는 구조를 이해하고 css를 응용해볼 수 있는 프로젝트였습니다.

sulwhasoo ㅣ 설화수

  • 페이지설계 : Web 100% Main

  • 제작기간 : 3일 퍼블리싱 100%

  • 웹 크기 : max-width: 1200px

    태블릿 크기 :max-width: 1000px

    모바일 크기 :max-width: 768px

  • 프로젝트 개요 :

    다양한 연령층을 가지고 있는 화장품 회사 중 설화수 홈페이지를 골라서 해보았습니다. 자바스크립트와 제이쿼리보다는 스와이퍼 플러그인을 이용하여 슬라이드를 구현하였습니다.

richmond ㅣ 리치먼드 평창

  • 페이지설계 : Web Main

  • 제작기간 : 8시간 퍼블리싱 100%

  • 웹 크기 : max-width: 100%

    태블릿 크기 :max-width: 1024px

  • 프로젝트 개요 :

    리치먼드평창 사이트는 웹사이트의 퍼블리싱 기능도 중요하지만 이미지로 인해 사이트에 더욱 생동감과 분위기를 살릴 수 있는 부분이 많았습니다. 특히 자바스크립트 기능없이도 마우스의 이벤트에 사용성을 더욱 보여줄 수 있어서 정적이지 않아 지루함이 없어서 구현해보았습니다.
